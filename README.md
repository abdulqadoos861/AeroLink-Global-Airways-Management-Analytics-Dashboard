📊 AeroLink Global Airways — Power BI Dashboard
📌 Project Overview

This project presents an end-to-end Power BI dashboard built for a simulated airline company, AeroLink Global Airways (AGAL). The dashboard analyzes flight operations, revenue performance, and employee data across three financial years (FY 2023–24 to FY 2025–26).

The goal is to transform raw data into meaningful business insights using data modeling, DAX calculations, and interactive visualizations.

📁 Dataset Description

The project uses multiple datasets:

Flight Sales Data — Seats sold by class for each flight
Flight Expenses — Operating and allocated costs per flight
Seat Pricing Details — Price per seat class
Location Data — Country and continent information
Employee Data — Employee details including passport expiry
⚙️ Key Features
📌 Data Cleaning & Transformation
Removed null rows/columns
Fixed data types and header issues in Power Query
📌 Data Modeling
Built relationships between tables (star schema approach)
Connected flights, locations, dates, and expenses
📌 DAX Calculations
Revenue (class-wise & total)
Total Cost and Profit
MTD (Month-to-Date) and QTD (Quarter-to-Date) metrics
📌 Interactive Dashboards
Multi-page report with navigation using bookmarks
Cross-page filtering using synced slicers
Dynamic visuals responding to user selections
